# CYBER GOMOKU - 赛博朋克五子棋

## 1. 项目概述

**项目名称:** CYBER GOMOKU
**项目类型:** 浏览器小游戏 / 人工智能应用
**项目简介:** 一款融合赛博朋克美学风格的五子棋对战游戏，支持三个难度等级的 AI 对手。
**目标用户:** 休闲游戏玩家，五子棋爱好者

## 2. 技术栈

- **前端框架:** 原生 HTML5 + CSS3 + Vanilla JavaScript
- **字体:** Google Fonts (Orbitron, Share Tech Mono)
- **部署方式:** 单文件部署，无需后端

## 5. AI 评分规则

AI 使用位置评分算法评估每个可能的落子位置，评分基于棋型的威胁程度。

| 棋型 | 描述 | 分数 |
|------|------|------|
| 五连 (FIVE) | 五子连珠，直接获胜 | 100000 |
| 活四 (LIVE_FOUR) | 两端开放的四连 | 10000 |
| 冲四 (RUSH_FOUR) | 一端封闭的四连 | 1000 |
| 活三 (LIVE_THREE) | 两端开放的活三 | 500 |
| 眠三 (SLEEP_THREE) | 一端封闭的三连 | 100 |
| 活二 (LIVE_TWO) | 两端开放的二连 | 50 |
| 眠二 (SLEEP_TWO) | 一端封闭的二连 | 10 |
| 活一 (LIVE_ONE) | 单子，一端开放 | 2 |

### 评分算法

1. **候选位置筛选:** 仅评估有邻近棋子的位置 (2格范围内)
2. **多方向评估:** 在四个方向 (水平/垂直/对角) 评估
3. **综合评分:** 位置总得分 = AI 棋型得分 + 玩家威胁得分
4. **搜索优化:**
   - 简单模式: 随机 + 少量候选
   - 中等模式: 全局评估最佳位置
   - 困难模式: Minimax 搜索树 + Alpha-Beta 剪枝
